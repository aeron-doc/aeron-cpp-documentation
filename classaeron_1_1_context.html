Detailed Description

This class provides configuration for the Aeron class via the Aeron::Aeron or Aeron::connect methods and its overloads. It gives applications some control over the interactions with the Aeron Media Driver. It can also set up error handling as well as application callbacks for connection information from the Media Driver.

Examples:
BasicPublisher.cpp, and BasicSubscriber.cpp.
